OpenWRT DNS

Download DNS list from github

curl -O https://raw.githubusercontent.com/felixonmars/dnsmasq-china-list/refs/heads/master/install.sh
chmod +x install.sh
./install.sh

Configure the list in DNS

Login the openwrt UI: Network -> DHCP and DNS -> Advanced Settings: Additional servers file:/etc/dnsmasq.d/accelerated-domains.china.114.114.114.114.conf Save and Apply. Note: it seems the dnsmasq takes time to load the config, wait for a few minutes and verify with dig. 
opkg install bind-dig
# dig 
dig @114.114.114.114 www.iqiyi.com
dig @8.8.8.8 www.iqiyi.com
dig www.iqiyi.com
