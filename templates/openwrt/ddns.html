OpenWRT DynamicDNS with DigitalOcean

Get Token from Digital Ocean

API -> Generate New Token

Get record id from DigitalOcean

ACCESS_TOKEN=
DOMAIN=nil.pp.ua
curl \
  --silent \
  -X GET \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $ACCESS_TOKEN" \
  "https://api.digitalocean.com/v2/domains/$DOMAIN/records"

Install software

System -> Software -> search and install ddns-scripts-digitalocean

Configure ddns

Services -> Dynamic DNS -> Add new services -> select digital ocean. Edit
  • Lookup hostname: nil.pp.ua
  • Domain: nil.pp.ua
  • Username: @
  • Password: from previous step
  • Optional Parameter: record id from previous step
