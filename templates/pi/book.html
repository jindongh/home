Day 8: Self hosted cloud storage for photos ith immich on Raspberry Pi 5

Start server

BOOK=/ssd/share/book
cat < docker-compose.yaml <<EOF
services:
    alist:
        image: 'xhofe/alist:latest'
        container_name: alist
        volumes:
            - ../data/alist:/opt/alist/data
            - /ssd/share/book:/book:ro
        network_mode: 'host'
        environment:
            - PUID=0
            - PGID=0
            - UMASK=022
        restart: unless-stopped
EOF
docker compose up -d

Access website

The job is running, now you can login from the browser, http://raspberrypi.local:5244 To add the book folder:
  1. Click Manage
  2. Click Storage
  3. Click Add
  4. Choose Local
  5. Mount Path: /pi
  6. Root folder path: /boot
  7. Root folder path: /boot
  8. Click Save

Access using Boox
